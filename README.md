# fundamental-
C++ Fundamental adalah kumpulan materi dasar pemrograman C++ yang dirancang untuk pemula maupun sebagai pengingat cepat bagi programmer berpengalaman. Repository ini berisi contoh kode, penjelasan konsep inti, dan latihan sederhana untuk memahami cara kerja bahasa C++ dari dasar hingga siap digunakan dalam pengembangan aplikasi nyata.
